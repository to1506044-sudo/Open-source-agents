# Agent S

## 项目描述

Agent S 是高性能开源Computer Use框架，像人类一样操作电脑，在OSWorld等基准上表现优秀。提供gui-agents库。

## 在Hermes Agent架构中的角色

Agent S 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供计算机操控能力。
