# GPT4All

![GPT4All](https://img.shields.io/badge/GPT4All-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[GPT4All](https://github.com/nomic-ai/gpt4all) 是无需联网、消费级硬件可跑的本地聊天应用。

**许可证**: MIT | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| 🏠 完全离线 | 无需联网使用 |
| 💻 消费级硬件 | 普通电脑可跑 |
| 🖥️ 桌面应用 | 开箱即用 |
| 📚 丰富模型 | 多种量化模型 |
