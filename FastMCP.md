# FastMCP

## 项目信息

- **GitHub**: [PrefectHQ/fastmcp](https://github.com/PrefectHQ/fastmcp)
- **分类**: MCP框架
- **语言**: Python
- **Stars**: 25k+

## 项目描述

FastMCP 是快速构建MCP服务器和客户端的Pythonic框架，提供简洁的装饰器API来定义工具和资源，是MCP开发的首选框架。

## 在Hermes Agent架构中的角色

FastMCP 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供MCP框架能力。
