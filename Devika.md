# Devika

![Devika](https://img.shields.io/badge/Devika-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[Devika](https://github.com/stitionai/devika) 是开源版Devin，自主软件工程师Agent。

**许可证**: MIT | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| 🤖 自主工程师 | 像Devin自主完成 |
| 🔍 浏览器搜索 | 自动搜索信息 |
| 📝 代码生成 | 自动编写调试 |
| 🖥️ Web界面 | 可视化操作 |
