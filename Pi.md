# Pi

![Pi](https://img.shields.io/badge/Pi-FF6B35?style=for-the-badge&logo=raspberrypi&logoColor=white)

## 简介

[Pi](https://github.com/earendil-works/pi) 是一个 **AI Agent 工具包**，提供统一的 LLM API、Agent 循环、TUI 界面和编程能力，让开发者能够快速构建和部署 AI 智能体应用。

⭐ **67.6k Stars** | 🍴 **4.2k Forks**

## 核心特性

| 特性 | 说明 |
|------|------|
| 🔗 统一 API | 统一的 LLM API 接口，支持多种模型 |
| 🔄 Agent 循环 | 内置 Agent 执行循环和状态管理 |
| 🖥️ TUI 界面 | 终端原生的交互式界面 |
| ✍️ 编程能力 | 强大的代码生成和执行能力 |
| 🔧 工具集成 | 丰富的工具和扩展能力 |
| 📦 模块化 | 高度模块化，易于扩展和定制 |

## 使用场景

- **AI 助手**：构建智能对话和任务执行助手
- **代码生成**：自动生成和执行代码
- **数据分析**：自动化数据处理和可视化
- **任务自动化**：将重复性工作交给 AI 自主完成
- **研究工具**：辅助学术研究和文献分析

## 快速开始

```bash
# 安装
pip install pi-ai

# 配置
export PI_API_KEY="your-api-key"

# 基本用法
pi "帮我分析这段代码"
pi "创建一个 Web 应用"
pi "解释这个算法"
```
