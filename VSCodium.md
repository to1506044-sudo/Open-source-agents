# VSCodium

![VSCodium](https://img.shields.io/badge/VSCodium-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[VSCodium](https://github.com/VSCodium/vscodium) 是去微软遥测的VS Code开源构建版。

**许可证**: MIT | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| 🔒 隐私保护 | 移除微软遥测 |
| 📦 VS Code等价 | 功能一致 |
| 🆓 完全免费 | MIT开源 |
| 🔌 插件兼容 | 兼容VS Code插件 |
