# LibreChat

![LibreChat](https://img.shields.io/badge/LibreChat-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[LibreChat](https://github.com/danny-avila/LibreChat) 是多模型聚合的开源ChatGPT替代。

**许可证**: MIT | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| 🔀 多模型聚合 | 一个界面多模型 |
| 🔌 丰富集成 | 数十种LLM后端 |
| 📝 多模态 | 图像/文件多模态 |
| 👥 多用户 | 团队协作使用 |
