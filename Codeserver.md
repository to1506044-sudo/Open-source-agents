# Codeserver

![Codeserver](https://img.shields.io/badge/Codeserver-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[Codeserver](https://github.com/coder/code-server) 是让VS Code跑在浏览器里，常被用作AI Agent沙盒IDE。

**许可证**: MIT | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| 🌐 浏览器IDE | 浏览器运行VS Code |
| 🐳 沙盒环境 | Agent执行沙盒 |
| 🏠 自托管 | 自主部署 |
| 👥 多用户 | 多用户同时使用 |
