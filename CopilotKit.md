# CopilotKit

![CopilotKit](https://img.shields.io/badge/CopilotKit-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[CopilotKit](https://github.com/CopilotKit/CopilotKit) 是将自定义AI Copilot嵌入React应用的开源框架。

**许可证**: MIT | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| ⚛️ React集成 | 深度React集成 |
| 🤖 自定义Copilot | 构建专属助手 |
| 📝 对话界面 | 内置聊天UI |
| 🔧 可定制 | 高度可定制 |
