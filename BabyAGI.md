## 核心特性

| 特性 | 说明 |
|------|------|
| 🎯 任务驱动 | 自主创建执行 |
| 🧠 记忆系统 | 向量数据库存储 |
| 🔄 自主循环 | 规划→执行→反馈 |
| 📖 教学经典 | Agent入门示例 |
