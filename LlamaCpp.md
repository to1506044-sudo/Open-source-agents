# LlamaCpp

![LlamaCpp](https://img.shields.io/badge/LlamaCpp-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[LlamaCpp](https://github.com/ggml-org/llama.cpp) 是C/C++实现的LLM本地推理引擎，生态基石。

**许可证**: MIT | **星标**: 数十万

## 核心特性

| 特性 | 说明 |
|------|------|
| ⚡ 极致性能 | C/C++原生实现 |
| 🖥️ 全平台 | CPU/GPU/嵌入式 |
| 📦 GGUF格式 | 定义量化标准 |
| 🌍 生态基石 | 众多项目基于此 |
