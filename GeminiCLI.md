# Gemini CLI

![Gemini CLI](https://img.shields.io/badge/Gemini_CLI-4285F4?style=for-the-badge&logo=google&logoColor=white)

## 简介

[Gemini CLI](https://github.com/google-gemini/gemini-cli) 是 Google 推出的 **终端原生 AI 编程助手**，基于强大的 Gemini 模型，让开发者直接在命令行中与 AI 协作完成编程任务。

## 核心特性

| 特性 | 说明 |
|------|------|
| 🖥️ 终端原生 | 直接在终端中运行，无缝融入开发工作流 |
| 🧠 代码理解 | 深度分析代码库结构和上下文 |
| ✍️ 代码生成 | 根据自然语言描述生成高质量代码 |
| 🔍 智能搜索 | 集成 Google 搜索能力，获取最新信息 |
| 🌐 多模态 | 支持文本、图像等多种输入形式 |
| 🔧 工具集成 | 丰富的工具和扩展能力 |

## 使用场景

- **快速开发**：用自然语言描述需求，自动生成代码
- **代码审查**：AI 辅助分析代码质量和潜在问题
- **重构优化**：大规模代码重构和性能优化
- **学习探索**：理解陌生代码库和技术栈
- **文档生成**：自动生成代码文档和注释

## 快速开始

```bash
# 安装
npm install -g @google/gemini-cli

# 配置 API Key
export GEMINI_API_KEY="your-api-key"

# 基本用法
gemini "创建一个 React 组件"
gemini "优化这段代码性能"
gemini "解释这段代码的作用"
```
