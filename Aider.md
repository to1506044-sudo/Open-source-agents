## 简介

[Aider](https://github.com/paul-gauthier/aider) 是Git 原生终端结对编程工具，每次修改自动commit。

**许可证**: Apache-2.0 | **星标**: ~41-47k

## 核心特性

| 特性 | 说明 |
|------|------|
| 📝 Git原生 | 自动commit |
| 🗣️ 结对编程 | 终端与AI对话 |
| 🔄 多模型 | Claude/GPT/DeepSeek |
| 📁 多文件 | 同时编辑多文件 |
