# Ollama

![Ollama](https://img.shields.io/badge/Ollama-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[Ollama](https://github.com/ollama/ollama) 是一条命令本地运行LLM的CLI工具，Agent本地化首选。

**许可证**: MIT | **星标**: ~162k

## 核心特性

| 特性 | 说明 |
|------|------|
| 🚀 一键运行 | 一条命令启动LLM |
| 📦 模型库 | 丰富预训练模型 |
| 🔌 API兼容 | 兼容OpenAI API |
| 🏠 完全本地 | 数据不出本地 |
