# Crawl4AI

## 项目信息

- **GitHub**: [unclecode/crawl4ai](https://github.com/unclecode/crawl4ai)
- **分类**: 网页爬取
- **语言**: Python
- **Stars**: 15k+

## 项目描述

Crawl4AI 是一个为AI优化的网页爬取工具，能够高效地从网页中提取结构化数据，支持异步爬取和智能解析。

## 在Hermes Agent架构中的角色

Crawl4AI 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供网页爬取能力。
