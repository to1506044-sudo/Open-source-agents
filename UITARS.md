# UI-TARS

## 项目信息

- **GitHub**: [bytedance/UI-TARS](https://github.com/bytedance/UI-TARS)
- **分类**: 视觉定位
- **语言**: Python
- **Stars**: 5k+

## 项目描述

UI-TARS 是字节跳动开发的GUI grounding视觉模型，专门做GUI元素定位和识别，可作为MiMo之外的辅助定位模型。

## 在Hermes Agent架构中的角色

UI-TARS 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供视觉定位能力。
