# Cursor Agent

![Cursor Agent](https://img.shields.io/badge/Cursor_Agent-7C3AED?style=for-the-badge&logo=cursor&logoColor=white)

## 简介

[Cursor Agent](https://github.com/civai-technologies/cursor-agent) 是 Cursor IDE 内置的 **AI 编程智能体**，能够自主理解代码库、规划任务并执行编码操作，是 Cursor 编辑器的核心 AI 能力。

## 核心特性

| 特性 | 说明 |
|------|------|
| 🧠 代码理解 | 深度分析项目结构和代码逻辑 |
| ✍️ 智能补全 | 基于上下文的高质量代码生成 |
| 🔧 自主编码 | 自动完成文件创建、修改、重构 |
| 🐛 问题修复 | 识别并修复代码中的 Bug |
| 📝 文档生成 | 自动生成代码文档和注释 |
| 💬 对话交互 | 通过自然语言与 AI 协作编程 |

## 使用场景

- **日常开发**：智能代码补全和生成
- **代码重构**：大规模代码优化和结构调整
- **Bug 修复**：快速定位和解决代码问题
- **新功能开发**：根据需求描述自动生成代码
- **代码审查**：AI 辅助代码审查和质量检查

## 快速开始

```bash
# Cursor Agent 内置于 Cursor IDE
# 下载 Cursor: https://cursor.sh

# 安装后直接使用
# 1. 打开项目文件夹
# 2. 使用 Cmd+K 生成代码
# 3. 使用 Cmd+L 打开 AI 对话
# 4. 使用 Cmd+I 启动 Agent 模式
```
