# Activepieces

## 简介

[Activepieces](https://github.com/activepieces/activepieces) 是开源自动化工作流平台，MCP/Agent集成。

**许可证**: MIT | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| 🔄 自动化 | 开源工作流 |
| 🤖 MCP集成 | 支持MCP协议 |
| 🔌 丰富连接器 | 多种应用连接 |
| 📝 代码友好 | 自定义代码步骤 |
