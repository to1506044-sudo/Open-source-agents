# 📖 About

随着大语言模型的发展，AI 正从 **Chat（聊天）** 进入 **Agent（智能体）** 时代。

未来的交互方式将从：

```
Human
   │
      ▼
      Software
      ```

      逐渐演变为：

      ```
      Human
         │
            ▼
            AI Agent
               │
                  ▼
                  Computer / Browser / Mobile / API / Robot
                  ```

                  用户只需要描述目标，AI Agent 将自动规划、执行并完成任务。

                  本仓库旨在打造一个**全球 AI Agent 开源项目导航**，帮助开发者快速了解和发现优秀项目。
