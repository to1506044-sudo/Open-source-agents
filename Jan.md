# Jan

![Jan](https://img.shields.io/badge/Jan-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[Jan](https://github.com/janhq/jan) 是完全离线优先的开源ChatGPT替代客户端。

**许可证**: AGPL-3.0 | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| 🏠 完全离线 | 无需联网使用 |
| 🖥️ 桌面应用 | 跨平台桌面客户端 |
| 📦 模型管理 | 内置模型下载 |
| 🔧 可扩展 | 支持扩展插件 |
