# PyAutoGUI

## 项目信息

- **GitHub**: [asweigart/pyautogui](https://github.com/asweigart/pyautogui)

- **语言**: Python
- **Stars**: 10k+

## 项目描述

PyAutoGUI 是最经典的Python跨平台GUI自动化库，用于精确控制鼠标位置、点击、拖拽、键盘输入、截屏等。许多desktop agent项目都依赖它。

## 在Hermes Agent架构中的角色

PyAutoGUI 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供计算机操控能力。
