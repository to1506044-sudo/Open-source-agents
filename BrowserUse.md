# Browser-Use

## 项目信息

- **GitHub**: [browser-use/browser-use](https://github.com/browser-use/browser-use)
- **分类**: 浏览器自动化
- **语言**: Python
- **Stars**: 20k+

## 项目描述

Browser-Use 是一个强大的浏览器自动化框架，通过视觉DOM树联合解析算法和高动态元素锚定技能，实现跨验证码和动态滑块的稳定操作。它能把视觉模型看到的图像像素点精准翻译成屏幕的绝对物理坐标。

## 在Hermes Agent架构中的角色

Browser-Use 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供浏览器自动化能力。
