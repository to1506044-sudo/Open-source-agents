# LaVague

## 项目信息

- **GitHub**: [lavague-ai/LaVague](https://github.com/lavague-ai/LaVague)
- **分类**: AI框架
- **语言**: Python
- **Stars**: 5k+

## 项目描述

LaVague 是大语言行动模型(LAM)框架，能把自然语言思维拆解并编译成一连串物理按键动作序列，实现长周期任务自动化。

## 在Hermes Agent架构中的角色

LaVague 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供AI框架能力。
