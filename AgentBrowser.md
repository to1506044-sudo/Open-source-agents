# Agent Browser

![Agent Browser](https://img.shields.io/badge/Agent_Browser-000000?style=for-the-badge&logo=vercel&logoColor=white)

## 简介

[Agent Browser](https://github.com/vercel-labs/agent-browser) 是 Vercel Labs 推出的 **面向 AI Agent 的浏览器自动化 CLI 工具**，让 AI 智能体能够像人类一样操控浏览器，完成网页交互任务。

⭐ **37.8k Stars** | 🍴 **2.4k Forks**

## 核心特性

| 特性 | 说明 |
|------|------|
| 🌐 浏览器自动化 | 像人类一样操控浏览器，点击、输入、导航 |
| 🤖 AI Agent 友好 | 专为 AI 智能体设计的 CLI 接口 |
| 📸 页面快照 | 自动捕获页面状态，供 AI 分析理解 |
| 🔧 可编程控制 | 支持 JavaScript 注入和页面操作 |
| 🐳 Docker 支持 | 容器化部署，轻松集成到 AI 工作流 |
| 📊 基准测试 | 完善的性能基准测试套件 |

## 使用场景

- **网页数据采集**：AI Agent 自动浏览网页，提取结构化数据
- **自动化测试**：智能体模拟用户操作，进行端到端测试
- **表单填写**：自动完成复杂的网页表单交互
- **内容审核**：AI 辅助检查网页内容质量和安全性
- **RPA 流程**：将重复的网页操作交给 AI 自动执行

## 快速开始

```bash
# 安装
npm install -g @anthropic-ai/agent-browser

# 基本用法
agent-browser navigate https://example.com
agent-browser screenshot
agent-browser click "button.submit"
```
