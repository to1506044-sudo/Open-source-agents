# Composio

## 项目信息

- **GitHub**: [composiohq/composio](https://github.com/composiohq/composio)
- **分类**: AI工具集成
- **语言**: Python
- **Stars**: 15k+

## 项目描述

Composio 是连接AI Agent和各种工具的万能胶水，完美支持MCP协议，能把Browser-Use、OpenInterpreter等工具统一打包封装，让Agent无缝调用。

## 在Hermes Agent架构中的角色

Composio 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供AI工具集成能力。
