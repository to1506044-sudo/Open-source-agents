# 🤖 Hermes Agent

> 全自主 AI 助手，支持桌面操作、代码开发、量化交易等任务

## 📌 项目信息

| 项目 | 内容 |
|------|------|
| **GitHub** | [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent) |
| **开发者** | Nous Research |
| **语言** | Python |
| **许可** | 开源 |

## ✨ 核心功能

- 🖥️ **桌面自动化** — 通过无障碍 API 控制 macOS 桌面
- 💻 **代码开发** — 自动编写、调试、测试代码
- 📊 **量化交易** — 支持加密货币交易策略执行
- 🌐 **浏览器控制** — 自动化网页操作
- 📱 **多平台集成** — 支持 Telegram、飞书、微信等
