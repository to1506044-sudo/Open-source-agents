# 🤖 Hermes Agent

> 全自主 AI 助手，支持桌面操作、代码开发、量化交易等任务

## 📌 项目信息

| 项目 | 内容 |
|------|------|
| **GitHub** | [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent) |
| **开发者** | Nous Research |
| **语言** | Python |
| **许可** | 开源 |

## ✨ 核心功能

- 🖥️ **桌面自动化** — 通过无障碍 API 控制 macOS 桌面，自动操作任意应用
- 💻 **代码开发** — 自动编写、调试、测试代码
- 📊 **量化交易** — 支持加密货币交易策略执行
- 🌐 **浏览器控制** — 自动化网页操作
- 📱 **多平台集成** — 支持 Telegram、飞书、微信等消息平台

## 🚀 为什么选择 Hermes Agent

Hermes Agent 是一个功能强大的 AI Agent 框架，可以：

1. 自主规划和执行复杂任务
2. 与多种外部工具和服务集成
3. 支持持久化记忆和跨会话学习
4. 通过 Skills 系统扩展功能
