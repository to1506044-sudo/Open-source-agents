# Theia

![Theia](https://img.shields.io/badge/Theia-Blue?style=for-the-badge&logo=github&logoColor=white)

## 简介

[Theia](https://github.com/eclipse-theia/theia) 是云端/桌面双用的开源IDE框架，可嵌入AI Agent。

**许可证**: EPL-2.0 | **星标**: 数万

## 核心特性

| 特性 | 说明 |
|------|------|
| ☁️ 云端+桌面 | 两种部署方式 |
| 🤖 Agent友好 | 可嵌入AI Agent |
| 🔧 高度可定制 | 模块化架构 |
| 🏛️ Eclipse基金会 | Eclipse治理 |
