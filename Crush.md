# Crush

![Crush](https://img.shields.io/badge/Crush-FF69B4?style=for-the-badge&logo=charm&logoColor=white)

## 简介

[Crush](https://github.com/charmbracelet/crush) 是 Charm 团队推出的 **AI 编程助手**，提供华丽的终端交互体验，让 AI 编程变得优雅而高效。

⭐ **26.1k Stars** | 🍴 **1.2k Forks**

## 核心特性

| 特性 | 说明 |
|------|------|
| 💎 精美界面 | Charm 风格的华丽终端 UI |
| 🧠 代码理解 | 深度分析代码库结构和上下文 |
| ✍️ 代码生成 | 根据自然语言描述生成高质量代码 |
| 🔄 智能重构 | 自动识别并优化代码结构 |
| 🎨 交互体验 | 流畅的交互式编程体验 |
| 🔧 工具集成 | 与 Charm 生态系统深度集成 |

## 使用场景

- **日常开发**：智能代码补全和生成
- **代码重构**：大规模代码优化和结构调整
- **Bug 修复**：快速定位和解决代码问题
- **新功能开发**：根据需求描述自动生成代码
- **代码审查**：AI 辅助代码审查和质量检查

## 快速开始

```bash
# 安装
go install github.com/charmbracelet/crush@latest

# 或使用 Homebrew
brew install charmbracelet/tap/crush

# 基本用法
crush "创建一个 Go Web 服务"
crush "优化这段代码性能"
crush "解释这个函数的作用"
```
