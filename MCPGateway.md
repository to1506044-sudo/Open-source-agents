# MCP Gateway

## 项目信息

- **GitHub**: [acehoss/mcp-gateway](https://github.com/acehoss/mcp-gateway)
- **分类**: MCP工具
- **语言**: TypeScript
- **Stars**: 133

## 项目描述

MCP Gateway 是灵活的MCP服务器网关，把STDIO MCP服务器桥接到HTTP+SSE和REST API，实现多实例和远程访问。

## 在Hermes Agent架构中的角色

MCP Gateway 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供MCP工具能力。
