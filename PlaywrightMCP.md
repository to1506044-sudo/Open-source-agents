# Playwright MCP

## 项目信息

- **GitHub**: [anthropics/playwright-mcp](https://github.com/anthropics/playwright-mcp)
- **分类**: MCP工具
- **语言**: TypeScript
- **Stars**: 3k+

## 项目描述

Playwright MCP 是基于 Playwright 的MCP服务器，提供浏览器自动化能力，支持跨浏览器的网页操作和测试。

## 在Hermes Agent架构中的角色

Playwright MCP 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供MCP工具能力。
