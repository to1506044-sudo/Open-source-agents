# Firecrawl

## 项目信息

- **GitHub**: [mendableai/firecrawl](https://github.com/mendableai/firecrawl)
- **分类**: 网页爬取
- **语言**: Python
- **Stars**: 20k+

## 项目描述

Firecrawl 是一个强大的网页爬取工具，能够将任何网站转换为LLM可用的结构化数据，支持JavaScript渲染和反爬虫机制。

## 在Hermes Agent架构中的角色

Firecrawl 是Hermes Agent全自主闭环架构中的重要组件，为Agent提供网页爬取能力。
