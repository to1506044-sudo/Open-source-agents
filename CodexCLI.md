# Codex CLI

![Codex CLI](https://img.shields.io/badge/Codex_CLI-412991?style=for-the-badge&logo=openai&logoColor=white)

## 简介

[Codex CLI](https://github.com/openai/codex) 是 OpenAI 推出的 **终端原生 AI 编程助手**，基于强大的 Codex 模型，让开发者直接在命令行中与 AI 协作完成编程任务。

## 核心特性

| 特性 | 说明 |
|------|------|
| 🖥️ 终端原生 | 直接在终端中运行，无缝融入开发工作流 |
| 🧠 代码理解 | 深度分析代码库结构和上下文 |
| ✍️ 代码生成 | 根据自然语言描述生成高质量代码 |
| 🔄 自动重构 | 智能代码优化和结构调整 |
| 🐛 Bug 修复 | 快速定位并修复代码问题 |
| 📝 提交管理 | 自动生成 Git 提交信息和 PR |

## 使用场景

- **快速开发**：用自然语言描述需求，自动生成代码
- **代码审查**：AI 辅助分析代码质量和潜在问题
- **重构优化**：大规模代码重构和性能优化
- **学习探索**：理解陌生代码库和技术栈
- **文档生成**：自动生成代码文档和注释

## 快速开始

```bash
# 安装
pip install openai-codex

# 配置 API Key
export OPENAI_API_KEY="your-api-key"

# 基本用法
codex "创建一个 Flask REST API"
codex "修复这个 bug"
codex "重构这段代码"
```
